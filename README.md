# ⏰ Digital_Clock

> A beautiful Python GUI Digital Clock application that displays real-time **12-hour format time with AM/PM**, along with **Date, Month, Year, and Day Name** using an attractive colorful interface. 🚀

---

# 🌟 About Project

**Digital_Clock** is a modern Python GUI application designed to show live digital time with a stylish and colorful user interface.

The application features:

✅ 12-Hour Time Format <br>
✅ AM / PM Indicator <br>
✅ Live Seconds Update <br>
✅ Date, Month & Year <br>
✅ Day Name Display <br>
✅ Attractive GUI Design <br>
